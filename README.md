# ChatRooms

## Description

ChatRooms is a real-time chat application. It was created with React in the frontend and Express, Node, and Socket.IO in the backend. Users can create rooms and join other rooms to chat instantly in real-time.

## Setup ‚è≥

> Follow these steps to setup this web application locally :

1. - [ ] Open your `Terminal`.
2. - [ ] Navigate to the directory where you will like to install the repo by running `cd FOLDER-NAME`
3. - [ ] Run `git clone git@github.com:khalilhamdii/chat-app-client.git` in the terminal to download.
4. - [ ] Run `cd chat-app-client` to enter directory.
5. - [ ] Run `npm install` to install node packages and dependencies.
6. - [ ] Run `npm start` in your browser to start the application.
